# EchoZone

EchoZone is a location-based, anonymous social media platform inspired by Yik Yak.
It allows users to post short messages visible only within a defined geographic radius.

## Core Features
- Anonymous posting
- Location-based feed
- Upvotes and downvotes
- Real-time updates
- Content moderation tools

## Planned Tech Stack
- Frontend: React / React Native
- Backend: Node.js + Express
- Realtime Data: Firebase / WebSockets
- Maps & Location: Mapbox
- Authentication: Anonymous / Device-based
